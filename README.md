# Anusha Nalla - Legal Professional Portfolio

A premium, interactive personal portfolio website built with modern web technologies to showcase my legal expertise, courtroom advocacy, research publications, and professional journey.

## 🚀 Features

- **Immersive Design:** Light-themed, high-contrast aesthetic with ambient gold lighting and Apple-tier glassmorphism effects.
- **Scrolly-telling Experience:** Buttery-smooth, hardware-accelerated scroll animations and transition effects guided by user interaction.
- **Interactive Legal Dossier:** Custom UI components including horizontal swipe galleries, interactive cards, and detailed modal views for various internships and publications.
- **Responsive Layout:** Fully optimized for desktops, tablets, and mobile devices with touch-friendly swipe gestures.
- **Contact Integration:** Functional, highly-styled contact form integrated via Web3Forms for direct email communication without needing server credentials, featuring an interactive monarch butterfly.
- **Analytics Ready:** Pre-configured with Vercel Analytics for privacy-friendly traffic monitoring.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Deployment & Analytics:** Vercel
